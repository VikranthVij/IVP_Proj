# Image Super Resolution using Bicubic Interpolation

## Overview
This project implements image upscaling using bicubic interpolation and evaluates performance using PSNR and SSIM.

## Features
- Bicubic interpolation
- Bilinear and Nearest Neighbor comparison
- Sharpening filter (enhancement)
- PSNR and SSIM evaluation
- Visualization of results

## Tech Stack
- Python
- OpenCV
- NumPy
- Matplotlib
- scikit-image

## How to Run
```bash
pip install -r requirements.txt
python main.py
